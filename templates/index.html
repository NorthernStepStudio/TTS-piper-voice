Piper TTS API

A simple REST API to synthesize natural, human-sounding speech from text. Stream raw audio or receive base64-encoded MP3 in JSON — perfect for AI agents, automation pipelines, and any app that needs a voice.

GET /api/v1/health Server health check

Returns server status and uptime. Use this to verify the API is reachable before making synthesis requests.

GET /api/v1/voices List all available voices

Returns all available voice IDs, names, genders, and style descriptions. Use the id field as the voice parameter in synthesis requests.

cURL
Python
JavaScript

POST /api/v1/synthesize Returns raw audio bytes

Synthesizes speech and streams back the audio file (MP3 or WAV). Best for saving audio to disk or piping to a media player.

Request Body (JSON)
text requiredstringText to synthesize. Max 5000 characters.
voice optionalstringVoice ID from /voices. Default: en-US-AvaNeural
rate optionalintegerSpeed % adjustment. Range −50 to +100. Default: 0
pitch optionalintegerPitch shift in Hz. Range −20 to +20. Default: 0
volume optionalintegerVolume % adjustment. Range −50 to +50. Default: 0
format optionalstring"mp3" or "wav". Default: "mp3"
cURL
Python
JavaScript
Node.js
curl -X POST {{ base_url }}/api/v1/synthesize \ -H "Content-Type: application/json" \ -d '{"text":"Hello world!","voice":"en-US-AvaNeural"}' \ --output speech.mp3

POST /api/v1/speak Returns JSON + base64 audio

Returns a JSON response with audio encoded as base64. Ideal for AI agents — no binary handling, includes request ID and generation metadata.

Request Body — same parameters as /synthesize
Response
{ "ok": true, "request_id": "f3a2b1c0-...", "audio": { "base64": "//NExAA...", "format": "mp3", "mime_type": "audio/mpeg", "size_bytes": 28416 }, "voice": { "id": "en-US-AvaNeural", "name": "Ava", "gender": "Female" }, "settings": { "rate": "+0%", "pitch": "+0Hz", "volume": "+0%" }, "meta": { "text_length": 12, "generation_seconds": 0.84 } }
cURL
Python (AI agent)
JavaScript
curl -X POST {{ base_url }}/api/v1/speak \ -H "Content-Type: application/json" \ -d '{"text":"Hello!","voice":"en-US-AvaNeural"}'
import requests, base64 result = requests.post( "{{ base_url }}/api/v1/speak", json={"text": "Task complete.", "voice": "en-US-AndrewNeural"} ).json() audio = base64.b64decode(result["audio"]["base64"]) with open("out.mp3", "wb") as f: f.write(audio) print(f"Done in {result['meta']['generation_seconds']}s")
const { audio } = await (await fetch("{{ base_url }}/api/v1/speak", { method: "POST", headers: { "Content-Type": "application/json" }, body: JSON.stringify({ text: "Hello!", voice: "en-US-EmmaNeural" }), })).json(); // Play directly from base64 in the browser new Audio(`data:${audio.mime_type};base64,${audio.base64}`).play();
