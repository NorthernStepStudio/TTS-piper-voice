Piper TTS API

A simple REST API to synthesize natural, human-sounding speech from text. Stream raw audio or receive base64-encoded MP3 in JSON — perfect for AI agents, automation pipelines, and any app that needs a voice.

Returns server status and uptime. Use this to verify the API is reachable before making synthesis requests.

GET /api/v1/voices List all available voices

Returns all available voice IDs, names, genders, and style descriptions. Use the id field as the voice parameter in synthesis requests.

POST /api/v1/synthesize Returns raw audio bytes

Synthesizes speech and streams back the audio file directly (MP3 or WAV). Best for saving audio to disk or piping to a media player.

Request Body (JSON)
text required string The text to synthesize. Max 5000 characters.
voice optional string Voice ID (see /voices). Default: en-US-AvaNeural
rate optional integer Speaking speed adjustment in %. Range −50 to +100. Default: 0
pitch optional integer Pitch shift in Hz. Range −20 to +20. Default: 0
volume optional integer Volume adjustment in %. Range −50 to +50. Default: 0
format optional string "mp3" or "wav". Default: "mp3"
cURL
Python
JavaScript
Node.js
curl -X POST {{ base_url }}/api/v1/synthesize \ -H "Content-Type: application/json" \ -d '{"text":"Hello world!","voice":"en-US-AvaNeural","rate":0}' \ --output speech.mp3
import requests response = requests.post( "{{ base_url }}/api/v1/synthesize", json={ "text": "Hello world!", "voice": "en-US-AvaNeural", "rate": 0, "pitch": 0, "volume": 0, } ) with open("speech.mp3", "wb") as f: f.write(response.content)
const response = await fetch("{{ base_url }}/api/v1/synthesize", { method: "POST", headers: { "Content-Type": "application/json" }, body: JSON.stringify({ text: "Hello world!", voice: "en-US-AvaNeural", rate: 0, pitch: 0, volume: 0, }), }); const blob = await response.blob(); // blob is audio/mpeg — play or save it
const fs = require("fs"); const res = await fetch("{{ base_url }}/api/v1/synthesize", { method: "POST", headers: { "Content-Type": "application/json" }, body: JSON.stringify({ text: "Hello!", voice: "en-US-BrianNeural" }), }); const buf = Buffer.from(await res.arrayBuffer()); fs.writeFileSync("speech.mp3", buf);
POST /api/v1/speak Returns JSON + base64 audio

Synthesizes speech and returns a JSON response with the audio encoded as base64. Ideal for AI agents — no binary handling needed, includes full metadata, request ID, and generation time.

Request Body — same as /synthesize
Response
{ "ok": true, "request_id": "f3a2b1c0-...", "audio": { "base64": "//NExAA...", ← decode & save as .mp3 "format": "mp3", "mime_type": "audio/mpeg", "size_bytes": 28416 }, "voice": { "id": "en-US-AvaNeural", "name": "Ava", ... }, "settings": { "rate": "+0%", "pitch": "+0Hz", "volume": "+0%" }, "meta": { "text_length": 12, "generation_seconds": 0.84 } }
cURL
Python (AI agent)
JavaScript
curl -X POST {{ base_url }}/api/v1/speak \ -H "Content-Type: application/json" \ -d '{"text":"Hello!","voice":"en-US-AvaNeural"}'
import requests, base64 # AI agent example: synthesize and save result = requests.post( "{{ base_url }}/api/v1/speak", json={ "text": "Task complete. The report has been generated.", "voice": "en-US-AndrewNeural", "rate": -10, # slightly slower } ).json() # Decode and save audio audio_bytes = base64.b64decode(result["audio"]["base64"]) with open("agent_output.mp3", "wb") as f: f.write(audio_bytes) print(f"Generated in {result['meta']['generation_seconds']}s")
const result = await (await fetch("{{ base_url }}/api/v1/speak", { method: "POST", headers: { "Content-Type": "application/json" }, body: JSON.stringify({ text: "Hello!", voice: "en-US-EmmaNeural" }), })).json(); // Play in browser directly from base64 const audio = new Audio(`data:audio/mpeg;base64,${result.audio.base64}`); audio.play();
